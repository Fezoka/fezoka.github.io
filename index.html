Marketplace Scout AI

Um ecossistema completo de RPA desenvolvido em Python que automatiza operações de arbitragem, compra e venda no Facebook Marketplace, integrando web scraping, IA e mensageria.

⚙️ Principais Funcionalidades

  • Motor de Arbitragem e Compras: Scraping inteligente de anúncios com foco em eletrônicos de alta liquidez; análise de mercado com Gemini para calcular ROI e gerar ranking de oportunidades; negociação automatizada via WhatsApp com abordagens persuasivas geradas por IA.
  • Motor de Vendas e Engajamento: Garimpo e entrada automática em grupos locais do Facebook; divulgação em massa de anúncios ativos respeitando limites diários e evitando bloqueios por duplicidade.
  • Dashboard Web em Tempo Real: Painel FastAPI para orquestrar robôs sem terminal; execução assíncrona com WebSockets, Threads e Subprocesses para rodar Selenium em segundo plano transmitindo logs ao front-end.

🛠️ Stack Tecnológico

  • Linguagem Core: Python.
  • Web Scraping & RPA: Selenium WebDriver, Webdriver Manager, técnicas de bypass de detecção (undetected-chromedriver).
  • Backend & API: FastAPI, Uvicorn, WebSockets.
  • IA: Google Gemini (engenharia de prompt para JSON e análise financeira).
  • Mensageria: Automação de WhatsApp Web via injeção de DOM.
  • Armazenamento: JSON (cache e histórico).

💡 Destaques Técnicos

  • Bypass de sistemas anti-bot com configurações específicas no Chrome Options e tempos randomizados.
  • Concorrência e assincronismo: filas de mensagens (queue) e processamento multithread para rodar automações bloqueantes dentro de uma aplicação assíncrona (FastAPI).
  • Engenharia de prompt dinâmica: injeção de histórico de preços via Clipboard para forçar respostas em JSON sem APIs pagas.

Sistema de Gestão de Laudos e Garantias (RMA Automator)

Uma aplicação web full-stack desenvolvida para digitalizar e gerenciar o ciclo de vida de garantias e devoluções (RMA) da empresa Royalstar, com geração automatizada de laudos em PDF e notificações via RPA.

⚙️ Principais Funcionalidades

  • Gestão do Ciclo de Vida de Garantias: Criação e acompanhamento de laudos com cálculo automático de status (Aberto, Em Processo, Fechado).
  • Geração Automatizada de PDFs: Conversão de laudos HTML para PDF profissional com navegador headless, incluindo fallback para Base64 de imagens.
  • Automação de E-mails via RPA: Robôs (Puppeteer) que simulam comportamento humano para acessar o webmail do Gmail e disparar notificações sobre criação/atualização de laudos.
  • Dashboards e Visualização: Gráficos dinâmicos apontando problemas mais frequentes, distribuição de status, top clientes e evolução mensal.
  • Upload Inteligente: Upload assíncrono com validação de formato e tamanho, preview em tempo real e visualização em tela cheia.
  • Controle de Acesso (RBAC): Perfis "Vendas" e "Técnico" com permissões distintas para abertura e preenchimento de chamados.

🛠️ Stack Tecnológico

  • Backend & API: Node.js com Express.js (RESTful).
  • RPA & Geração de Documentos: Puppeteer (Headless Chrome) para automação web e renderização de PDFs.
  • Frontend: HTML5, Vanilla JavaScript (Fetch API), CSS customizado e Bootstrap 5 (SPA).
  • Visualização de Dados: Chart.js.
  • Armazenamento: JSON (NoSQL-like) e Multer para upload de arquivos.
  • Autenticação: Login customizado com sessão via Tokens e LocalStorage.

💡 Destaques Técnicos

  • RPA inovador para mensageria: script que contorna limitações SMTP abrindo instâncias do Chrome em background, navegando pelo DOM do Gmail, preenchendo templates e enviando.
  • Resolução de problemas de renderização em PDFs: algoritmo de fallback que converte imagens para Base64 ou URLs absolutas em tempo de execução.
  • Arquitetura SPA sem frameworks pesados: manipulação da DOM com Vanilla JS, componentes modais responsivos e tratamento assíncrono rigoroso para uploads pesados.

Sistema de Gestão e Auditoria 5S Corporativo

Uma aplicação web full-stack desenvolvida para digitalizar, padronizar e automatizar o controle e as auditorias da metodologia 5S em um ambiente corporativo/industrial, substituindo controles manuais por dashboards interativos.

⚙️ Principais Funcionalidades

  • Auditorias Inteligentes: Checklists estruturados para áreas administrativas e operacionais ("5S Produtivo"), com respostas padronizadas (OK, NOK, N/A) e cálculo automático de pontuação.
  • Painel de Melhoria Contínua: Sistema integrado de Planos de Ação que sinaliza áreas com percentual crítico (abaixo de 60%) e gerencia prazos de conclusão.
  • Dashboards e Visualização: Gráficos interativos de evolução temporal (linhas) e perfis de conformidade (radar), tendência de desempenho e média geral.
  • Motor Automático de Alertas: Rotina em segundo plano que verifica vencimentos diariamente, enviando notificações na plataforma e e-mails automáticos sobre auditorias atrasadas e planos pendentes.
  • Painel Administrativo: Área restrita para gestores criarem cronogramas, gerenciarem setores, analisarem rankings automáticos (3 melhores/piores), gráficos agregados e comunicação em massa.
  • Autenticação e Segurança: Login seguro com hashes de senha, validação de e-mails autorizados e fluxo de cadastro com MFA via código numérico por e-mail.

🛠️ Stack Tecnológico

  • Backend: Python com Flask.
  • Banco de Dados: SQLite com migrações automáticas.
  • Frontend: HTML5, CSS customizado, Bootstrap 5.
  • Visualização de Dados: Chart.js (Doughnut, Radar, Linhas, Barras).
  • Automação e Threads: threading, smtplib, email.mime para concorrência e mensageria SMTP.

💡 Destaques Técnicos

  • Execução concorrente para tarefas agendadas: threading.Thread(daemon=True) acoplada ao Flask para varreduras diárias sem bloquear o Event Loop.
  • Extração avançada de KPIs via SQL: queries com LEFT JOINs, GROUP BY e COALESCE para cruzar tabelas e montar rankings de desempenho em tempo real.
